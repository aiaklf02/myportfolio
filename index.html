Ronda Moroccan card Game

Ronda is a digital adaptation of the traditional Moroccan card game, developed using the Qt 6 framework. The backend, implemented in C++, handles all core mechanics, including rule enforcement, turn management, and game state processing. The interface is built with QML, allowing dynamic rendering and real-time updates triggered by signals emitted from the C++ layer. The interplay between QML and C++ ensures a responsive experience and clean separation between logic and presentation, highlighting effective use of Qt’s modular architecture.

Academic Administration System

A university management web application developed in PHP with Laravel for the backend and native HTML, CSS, and JavaScript for the frontend. The platform enables complete management of students, courses, and teaching staff through a user-friendly interface. Its structure ensures secure data processing, organized workflows, and modular features tailored to academic administration without relying on frontend frameworks.

Campus Global

It is a web platform developed in Python using the Django framework. It provides an interactive environment where academic professionals can connect, collaborate, and share resources. The platform includes a virtual classroom for accessing courses, video lessons, and assessments, as well as a dedicated space for professional networking and community interaction within the academic sector.

Library Management mobile App

A mobile application designed with Flutter and Dart to modernize library services. It offers a complete set of features for book reservations, borrowing, and resource tracking. The interface provides a smooth user experience, allowing users to search for books, check real-time availability, manage their loans, and access key library services through an intuitive and modern design.

Parkinson Disease Classification

This project applies comprehensive data processing and feature engineering on vocal datasets to enhance Parkinson’s disease detection. An Ensemble Feature Selection Algorithm (EFSA) is used to identify the most relevant features. Multiple models including Logistic Regression, Random Forest, and LightGBM are developed from scratch, both with and without automated feature extraction. The project demonstrates a full end-to-end machine learning workflow, from raw data processing to feature selection and model optimization, supporting accurate early diagnosis for telemedicine applications.

The MLhub

Developed in Python using Django, scikit-learn, and TensorFlow, this application enables end-to-end machine learning workflows. It supports dataset import, preprocessing, visualization, and selection of ML algorithms, including Random Forest, LightGBM, Naïve Bayes, SVM, Logistic Regression, RNN, and KMeans. The platform also provides performance metrics and result visualization, allowing users to evaluate and compare model effectiveness in a user-friendly interface.

Real-Time Analysis of Amazon Reviews

A scalable Big Data system designed to scrape Amazon Music New Releases album reviews and analyze them in real time. The architecture is built with Kafka for streaming, Apache Spark MLlib for sentiment analysis, and MongoDB for storage. The pipeline is deployed using a containerized microservices architecture with Docker Compose. The data is streamed through Kafka, processed instantly with Spark, and stored in MongoDB. A Flask backend exposes processed insights, while a React dashboard with WebSocket integration delivers live sentiment visualization and access to historical data. The platform supports real-time analytics, data exporting, and continuous monitoring of user opinions.

CancerUX Intelligent Cancer Treatment Assistant

An NLP-based application built with a fine-tuned BioBERT model and a Flask interface. The system answers questions related to cancer treatments and side effects using specialized medical data, providing accurate, domain-specific responses to support decision-making and patient guidance.

FruitEye AI-Powered Rotten Fruit Detection System

An end-to-end intelligent application designed to automate the detection and classification of fresh and spoiled fruits. The project includes the full pipeline: image annotation, dataset construction, preprocessing, and model training. A Convolutional Neural Network based on MobileNetV2 is used for image classification, while YOLOv8 enables real-time object detection with high accuracy. The system is integrated into an interactive Streamlit interface, allowing users to upload images, visualize predictions, and test detection in real time. The solution demonstrates expertise in computer vision, deep learning, dataset engineering, and model deployment, offering applications in food quality control, agriculture, and supply chain automation.
